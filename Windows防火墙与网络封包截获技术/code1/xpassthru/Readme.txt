[sample]	调试信息实例
[setup]		编译好的驱动程序和用来安装的相关文档


测试方法：	
---------------------------------------------------------
1．	打开网卡连接属性面板
2．	按“安装”按钮，弹出“选择网络组件类型”对话框
3．	选中“服务”后按“添加按钮”，弹出“选择网络服务”对话框
4．	按“从磁盘安装” 按钮，弹出“从磁盘安装”对话框
5．	按“浏览”按钮，弹出“查找文件”对话框
6．	通过对话框找到本书配套光盘的setup目录下的xpassthru.inf文件，
	按“打开”按钮后弹出“选择网络服务”对话框
7．	按“确定”后系统便会自动复制所需文件，安装完成后回到第一步的界面，
	上面会多出一个“Xstudio Sample Filter”的服务，这就是Xpassthru。
8．	打开dbgview.exe(配书盘没有提供)，然后通过网上邻居访问其它计算机，
	这时会有调试信息输出到dbgview的界面上。
	例程中给出一个SampleLog.txt的文件，这个文件里是一个调试信息的样本。
9．	卸载。在第一步的界面上选中“Xstudio Sample Filter”后按“卸载”按钮
	完成Xpassthru的卸载。
